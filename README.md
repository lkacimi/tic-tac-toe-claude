## How to Play

1. Click on any empty square to place your mark (X or O)
2. Players alternate turns
3. First player to get 3 in a row (horizontally, vertically, or diagonally) wins
4. Use the move history on the right to jump back to any previous game state
5. The game detects draws when the board is full with no winner
